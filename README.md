# QUCT: Criticality Parameter gamma* for Riemann Zeros

This repository contains the reference implementation of QUCT (Qadmon Universal Criticality Theory) for computing the dimensionless criticality parameter gamma* from a variational stability functional F(Î³), and comparing it with the statistics of Riemann zeta zeros.

- Core code: `src/quct_gamma_root.py`
- Minimal LaTeX paper: `docs/quct_gamma_star.tex`
